prepare_data module¶

import_clean_data(csv_file_path, csv_file_save_to_path, threshold)¶

Import, clean and save CSV file.

Parameters

  • csv_file_path – Path of CSV file containing relevant data

  • csv_file_save_to_path – Path of result CSV file

  • threshold – Decimal value, representing the maximum NaN percentage of a row. If a row has more NaN values than the threshold, it will be dropped.
